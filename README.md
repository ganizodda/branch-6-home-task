# Food Delivery App – UI/UX Design

## Overview

This project is a modern **Food Delivery Mobile Application UI/UX Design** created in Figma.
The design focuses on delivering a clean, intuitive, and user-friendly experience for ordering food online.

The application includes complete user flows for:

* Authentication
* Home & discovery
* Restaurant browsing
* Food details
* Cart & checkout
* Order tracking
* Profile management

The interface is designed with modern UI principles, reusable components, and scalable layouts suitable for both iOS and Android platforms.

---

The primary goals of this project are:
